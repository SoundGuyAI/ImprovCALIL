# Israeli Improv Resources

Preliminary link list for organizers, schools, troupes, and community groups in Israel. Useful when seeding the calendar directory, verifying submissions, or finding event sources.

**Status:** Compiled from public web search (May 2026). Links and schedules change — verify before publishing.

| **Improvise.co.il**                          | Workshops (beginners → advanced), weekly improv group, corporate training. Short-form focus; Rod Ben Zeev–associated content on corporate workshops.         | [improvise.co.il](https://www.improvise.co.il/) · [Weekly group](https://www.improvise.co.il/improv-group) · [Workshops](https://www.improvise.co.il/workshops)                                                                                                                                                                                              |
